# Panduan Penggunaan Website AKAPRO

## Untuk Pengunjung

### 1. Melihat Program Pelatihan
1. Klik menu "Jasa Pelatihan" di beranda
2. Lihat 9 program pelatihan yang tersedia
3. Klik kartu pelatihan untuk info lebih lanjut
4. Klik icon WhatsApp untuk menghubungi CS

### 2. Melihat Jadwal Pelatihan
1. Klik menu "Jadwal Pelatihan" di beranda
2. Lihat jadwal pelatihan yang akan datang
3. Klik "Unduh TOR" untuk download Term of Reference dalam format PDF
4. Klik "Daftar Sekarang" untuk registrasi
5. Isi form registrasi lengkap
6. Lakukan pembayaran sesuai instruksi
7. Klik "Konfirmasi Pendaftaran"

### 3. Submit Artikel
1. Klik menu "Artikel" di beranda
2. Pilih tab "Submit Artikel"
3. Isi form lengkap:
   - Data pribadi (nama, email, instansi, alamat, no. WA)
   - Data artikel (judul, kategori, abstrak)
   - Upload file PDF artikel
4. Klik "Submit for Review"
5. Tunggu email konfirmasi dari tim editor

### 4. Membaca Artikel/Jurnal
1. Klik menu "Artikel" di beranda
2. Pilih tab "Daftar Jurnal"
3. Gunakan search bar untuk mencari artikel
4. Klik "Full Text PDF" untuk download
5. Klik "Cite" untuk mendapatkan format sitasi

### 5. Menghubungi CS
**Cara 1: Via Navigation Dock**
1. Klik icon WhatsApp hijau di bagian bawah layar
2. Template pesan otomatis akan muncul
3. Edit sesuai kebutuhan Anda
4. Kirim pesan

**Cara 2: Via Halaman Bantuan**
1. Klik menu "Pusat Bantuan"
2. Klik "Chat via WhatsApp"
3. Atau pilih salah satu quick action:
   - Informasi Aplikasi
   - Informasi Pelatihan
   - Submit Artikel
   - Mitra Kerjasama

### 6. Informasi Aplikasi PINTAR UC
1. Klik menu "Produk Aplikasi" di beranda
2. Pilih "PINTAR UC"
3. Scroll untuk melihat:
   - Latar belakang
   - Manfaat
   - Target pengguna
4. Klik "Ajukan Demo Sekarang" untuk request demo

1. Login ke dashboard admin
2. Pilih tab "Pelatihan"
3. **Tambah Pelatihan:**
   - Klik "Tambah Pelatihan"
   - Isi form lengkap
   - Klik "Simpan"
4. **Edit Pelatihan:**
   - Klik icon pensil pada pelatihan yang ingin diedit
   - Update data
   - Klik "Simpan"
5. **Hapus Pelatihan:**
   - Klik icon trash pada pelatihan yang ingin dihapus
   - Konfirmasi penghapusan

### Mengelola Artikel
1. Login ke dashboard admin
2. Pilih tab "Artikel"
3. Lihat daftar artikel yang disubmit
4. **Approve Artikel:**
   - Klik "Approve" pada artikel yang layak publish
   - Artikel akan muncul di halaman publik
5. **Reject Artikel:**
   - Klik "Reject" pada artikel yang tidak memenuhi syarat
   - Artikel tidak akan dipublikasikan

### Mengelola Konten
1. Login ke dashboard admin
2. Pilih tab "Konten"
3. Edit konten yang tersedia:
   - Nomor WhatsApp CS
   - Alamat kantor
   - Email kontak
   - dll
4. Klik "Simpan Perubahan"

### Logout
1. Klik tombol "Logout" di pojok kanan atas
2. Anda akan kembali ke halaman login

## Troubleshooting

### Masalah Umum

**1. Tidak bisa submit artikel**
- Pastikan semua field wajib (*) sudah diisi
- Cek ukuran file PDF tidak melebihi 10MB
- Pastikan koneksi internet stabil

**2. WhatsApp tidak terbuka**
- Pastikan WhatsApp terinstall di device
- Cek nomor WhatsApp sudah benar (085726112001)
- Coba refresh halaman

**3. PDF TOR tidak terdownload**
- Pastikan browser mengizinkan download
- Cek popup blocker tidak aktif
- Coba browser lain

**4. Admin tidak bisa login**
- Pastikan email dan password benar
- Cek koneksi internet
- Clear browser cache
- Hubungi super admin jika masih bermasalah
